data-toggle.js

SSR-uyumlu, hafif ve esnek bir toggle yöneticisi. Tek bir data-toggle attribute ile class toggle, scope, scroll lock, media query, focus ve daha fazlası.

İçindekiler

  • self — Kendine Toggle
  • target — Hedef Eleman
  • scope — Grup İçinde Tekil
  • lock-scroll — Kaydırma Kilidi
  • media — Media Query
  • delay — Gecikme
  • focus — Otomatik Odaklanma
  • outside-close — Dışarı Tıklama
  • Birleşik Örnekler

self

Temel

Tıklanan elemana toggle class'ı ekler/kaldırır. En basit kullanım şeklidir — buton kendi üzerinde class değiştirir.

data-toggle="self"
← Butona tıkla, .toggle class'ı eklenip kalkacak
HTML 
<button class="btn" data-toggle="self">
   Tıkla — Toggle Et
</button>
Nasıl çalışır? self parametresi, tıklanan elemanın kendisine .toggle class'ı ekler. Tekrar tıklanırsa class kaldırılır.

target:<selector>

Temel

Belirtilen CSS seçiciye sahip elemanlara toggle class'ı ekler/kaldırır. Buton başka bir öğeyi kontrol eder.

data-toggle="target:#my-panel"
📦 Hedef Panel — .toggle class'ı bu elemana eklenir
HTML 
<button data-toggle="target:#my-panel">Paneli Aç/Kapa</button>

<div id="my-panel" class="target-box">
   Hedef Panel
</div>
Çoklu hedef: Seçici birden fazla elemanla eşleşirse hepsine toggle uygulanır. Örneğin target:.card tüm .card elemanlarını etkiler.

scope:<selector>

Gelişmiş

Bir grup içinden sadece birinin aktif olmasını sağlar. Önce scope'taki tüm elemanlardan class kaldırılır, sonra seçilen elemana eklenir. Tab, akordeon gibi yapılar için idealdir.

data-toggle="target:#panel-1, scope:.tab-panel"
📄 Genel — Bu sekme varsayılan olarak aktif. Scope sayesinde diğer sekmeler kapanır.
⚙️ Ayarlar — Uygulama yapılandırma seçenekleri burada yer alır.
👤 Profil — Kullanıcı bilgileri ve tercihler bu sekmede.
HTML 
<!-- Butonlara tab-panel eklenir, scope hepsini temizler -->
<button class="tab-btn tab-panel toggle"
   data-toggle="self, target:#tab-1, scope:.tab-panel">
   Genel
</button>
<button class="tab-btn tab-panel"
   data-toggle="self, target:#tab-2, scope:.tab-panel">
   Ayarlar
</button>

<!-- Paneller -->
<div id="tab-1" class="tab-panel toggle">Genel içerik</div>
<div id="tab-2" class="tab-panel">Ayarlar içerik</div>
Davranış farkı: Scope kullanıldığında toggle yerine "birini aç, geri kalanını kapat" mantığı çalışır. Zaten aktif olan elemana tıklanırsa class kaldırılır (hiçbiri aktif olmaz).
Önemli: Butonlara da tab-panel class'ı eklenir. Böylece scope:.tab-panel hem butonlardan hem panellerden .toggle class'ını kaldırır, ardından sadece tıklananla ilişkili elemanlara ekler.

lock-scroll

Yardımcı

Toggle aktif olduğunda belirtilen elemana (varsayılan: body) overflow-hidden class'ı ekleyerek sayfa kaydırmasını engeller. Modal ve sidebar gibi overlay bileşenler için kullanılır.

data-toggle="target:#modal, lock-scroll"
Bu alan kaydırılabilir bir kutudur. Butona tıklayarak kaydırmayı kilitleyin.

Lorem ipsum dolor sit amet, consectetur adipiscing elit. Sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris.

Duis aute irure dolor in reprehenderit in voluptate velit esse cillum dolore eu fugiat nulla pariatur. Excepteur sint occaecat cupidatat non proident.

Sed ut perspiciatis unde omnis iste natus error sit voluptatem accusantium doloremque laudantium.
HTML 
<!-- body'ye overflow-hidden ekler (varsayılan) -->
<button data-toggle="target:#modal, lock-scroll">
   Modal Aç
</button>

<!-- Belirli bir elemana overflow-hidden ekler -->
<button data-toggle="target:#panel, lock-scroll:#scroll-area">
   Scroll Kilitle
</button>
Varsayılan: Seçici belirtilmezse (lock-scroll) body elemanına uygulanır. Özel bir hedef için lock-scroll:#my-container kullanın.

media:(<query>)

Koşullu

Toggle işlemi yalnızca belirtilen media query eşleştiğinde çalışır. Responsive davranışlar için kullanılır — örneğin sadece mobilde açılan menüler.

data-toggle="target:#mobile-menu, media:(max-width: 768px)"
Pencere genişliği: px
HTML 
<!-- Sadece 768px ve altında çalışır -->
<button data-toggle="target:#mobile-menu, media:(max-width: 768px)">
   Menü
</button>

<!-- Sadece 769px ve üstünde çalışır -->
<button data-toggle="target:#sidebar, media:(min-width: 769px)">
   Sidebar
</button>
window.matchMedia: Standart CSS media query sözdizimi kullanılır. Pencere boyutunu değiştirerek yukarıdaki butonların hangisinin çalıştığını test edebilirsiniz.

delay:<ms>

Zamanlama

Toggle işlemini belirtilen süre (milisaniye) kadar geciktirir. CSS animasyonları veya geçiş efektleri için kullanışlıdır. Varsayılan gecikme 16ms'dir.

data-toggle="target:#panel, delay:500"
16ms
300ms
1000ms
HTML 
<!-- Varsayılan 16ms gecikme -->
<button data-toggle="target:#panel">Anında</button>

<!-- 300ms sonra toggle -->
<button data-toggle="target:#panel, delay:300">Gecikmeli</button>

<!-- 1 saniye sonra toggle -->
<button data-toggle="target:#panel, delay:1000">Çok Gecikmeli</button>
Timeout temizleme: Aynı butona gecikme tamamlanmadan tekrar tıklanırsa eski timeout iptal edilir ve yeni bir zamanlayıcı başlar (debounce benzeri davranış).

focus

Erişilebilirlik

Toggle çalıştıktan sonra hedef eleman içindeki ilk odaklanabilir öğeye (input, textarea, select vb.) otomatik olarak focus verir. Arama kutuları ve formlar için idealdir.

data-toggle="target:#search-box, focus"
HTML 
<button data-toggle="target:#search-box, focus">
   🔍 Aramayı Aç
</button>

<div id="search-box">
   <input type="text" placeholder="Arama yapın..." />
</div>
Odaklanabilir elemanlar: input, textarea, select ve [tabindex] attribute'una sahip elemanlar hedef alınır. Yalnızca tek hedef varsa çalışır.

outside-close

Etkileşim

Hedef eleman açıkken dışına tıklanırsa otomatik olarak kapatılır. Dropdown menüler, popup'lar ve tooltip'ler için idealdir.

data-toggle="target:#dropdown, outside-close"
📁 Dosyalar ⚙️ Ayarlar 👤 Profil 🚪 Çıkış
← Menüyü aç, sonra dışına tıkla
HTML 
<div class="dropdown-wrapper">
   <button data-toggle="target:#dropdown, outside-close">
      Menüyü Aç ▾
   </button>

   <div id="dropdown" class="dropdown-menu">
      <a href="#">Dosyalar</a>
      <a href="#">Ayarlar</a>
   </div>
</div>
Dinleyici temizliği: Hedef kapandıktan sonra document üzerindeki click listener otomatik olarak kaldırılır, gereksiz dinleyici birikmesi olmaz.

📋 Akordeon

data-toggle, HTML elemanlarına class toggle işlevi ekleyen hafif bir JavaScript kütüphanesidir. SSR uyumludur ve MutationObserver ile dinamik elemanları da destekler.
scope parametresi, bir grup eleman içinde sadece birinin aktif olmasını sağlar. Yeni bir eleman aktifleştirildiğinde diğerleri otomatik olarak kapanır.
Script, window ve document nesnelerinin varlığını kontrol eder. Sunucu tarafında çalıştırıldığında hiçbir işlem yapmaz, bu sayede SSR ortamlarında hata vermez.
HTML 
<!-- scope ile sadece bir akordeon açık kalır -->
<button data-toggle="target:#acc-1, scope:.accordion-body.toggle">
   Başlık 1
</button>
<div id="acc-1" class="accordion-body">İçerik 1</div>

<button data-toggle="target:#acc-2, scope:.accordion-body.toggle">
   Başlık 2
</button>
<div id="acc-2" class="accordion-body">İçerik 2</div>

🏷️ Data Attribute Seçici

ℹ️ Bilgi: Sistem normal çalışıyor. Tüm servisler aktif durumda.
⚠️ Uyarı: Disk kullanımı %85'e ulaştı. Temizlik önerilir.
🚨 Hata: Veritabanı bağlantısı zaman aşımına uğradı!
HTML 
<!-- [data-attribute] seçicilerle scope kullanımı -->
<button data-toggle="target:[data-panel='info'], scope:[data-panel]">
   Bilgi
</button>
<button data-toggle="target:[data-panel='warning'], scope:[data-panel]">
   Uyarı
</button>
<button data-toggle="target:[data-panel='error'], scope:[data-panel]">
   Hata
</button>

<!-- data-panel attribute'lu paneller -->
<div data-panel="info">ℹ️ Bilgi mesajı</div>
<div data-panel="warning">⚠️ Uyarı mesajı</div>
<div data-panel="error">🚨 Hata mesajı</div>
Attribute seçiciler: target ve scope parametreleri standart CSS seçicilerini desteklediği için [data-*] attribute seçicileri de kullanılabilir. Class bağımlılığı olmadan tamamen data attribute'larla çalışan yapılar kurabilirsiniz.

Mimari ve Teknik Detaylar

Çalışma Akışı 
  1.  dataToggle('toggle') çağrılır
         │
  2.  Mevcut [data-toggle] elemanları bulunur → bindToggle()
         │
  3.  MutationObserver başlatılır (dinamik elemanlar için)
         │
  4.  Kullanıcı tıklar → click event tetiklenir
         │
  5.  parseArgs() → data-toggle değeri parse edilir (cache'lenir)
         │
  6.  Media query kontrolü → eşleşmezse iptal
         │
  7.  Scope varsa → scope elemanlarından class kaldır
         │
  8.  Delay ile → classList.toggle() uygula
         │
  9.  Ek özellikler: lock-scroll, focus, outside-close
JavaScript — Başlatma 
// ES Module import
import { dataToggle } from './data-toggle.js'

// Varsayılan class: 'toggle'
dataToggle()

// Özel class adı kullanma
dataToggle('active')
WeakMap & Cache: Timeout ID'leri WeakMap ile eleman bazında saklanır (bellek sızıntısı önlenir). Parse sonuçları Map ile cache'lenerek tekrarlı parse işlemleri engellenir.
MutationObserver: DOM'a sonradan eklenen elemanlar otomatik olarak tespit edilir ve bindToggle() ile bağlanır. SPA ve dinamik içerik yükleme senaryolarında sorunsuz çalışır.
SSR Güvenliği: window ve document kontrolü sayesinde sunucu tarafında (Node.js) hata vermez.
